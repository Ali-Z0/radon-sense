# Radon Sense — Landing Page

Landing page du projet **Radon Sense**, développée dans le cadre d’un projet MVP.

Radon Sense vise à rendre le risque radon plus visible, compréhensible et actionnable.  
Cette page présente le problème, la proposition de valeur, les fonctionnalités envisagées et permet aux personnes intéressées de laisser leurs coordonnées via un formulaire de contact.

---

## Objectif de la landing page

La landing page a été créée pour soutenir la phase MVP du projet.  
Elle permet de :

- présenter rapidement le problème du radon ;
- expliquer la promesse de Radon Sense ;
- montrer une première vision de l’interface utilisateur ;
- tester l’intérêt du public cible ;
- collecter des contacts via un formulaire ;
- disposer d’un support simple pour présenter le projet.

L’objectif n’est pas de fournir un produit final, mais de valider la compréhension du besoin et l’intérêt pour la solution proposée.

---

## Stack technologique

Le projet utilise une stack web légère et rapide :

- **React** pour la construction de l’interface ;
- **Vite** pour l’environnement de développement et le build ;
- **Tailwind CSS** pour le style et la mise en page ;
- **Web3Forms** pour le formulaire de contact sans backend dédié ;
- **GitHub Pages** pour le déploiement.

---

## Origine du template

Le projet est basé sur le template officiel **React + Vite** généré avec Vite.  
Ce template fournit une base minimale avec React, Vite, le hot module replacement et une configuration ESLint.

La structure initiale a ensuite été adaptée pour créer une landing page complète dédiée à Radon Sense :

- remplacement du contenu générique par le contenu du projet ;
- création d’une identité visuelle propre ;
- ajout des sections problème, fonctionnalités, bénéfices et FAQ ;
- ajout d’un formulaire de contact ;
- adaptation du style avec Tailwind CSS.

---

## Utilisation de l’intelligence artificielle

L’intelligence artificielle générative a été utilisée comme outil d’aide à la création de cette landing page.

---

## Sections principales

La landing page contient les sections suivantes :

- **Hero section** : présentation courte de Radon Sense et appel à l’action ;
- **Problème** : explication du risque radon et de sa difficulté d’interprétation ;
- **Fonctionnalités** : mesure continue, historique, alertes et recommandations ;
- **Valeur utilisateur** : comprendre, prévenir et décider ;
- **Fonctionnement** : parcours en trois étapes — mesurer, visualiser, agir ;
- **Réassurance** : éléments de confiance autour de la prévention du radon ;
- **Formulaire de contact** : collecte de contacts intéressés ;
- **FAQ** : réponses simples aux questions fréquentes.

## Installation

Installer les dépendances :

```bash
npm install
````

Lancer le serveur de développement :

```bash
npm run dev
```

Construire la version de production :

```bash
npm run build
```

Prévisualiser le build :

```bash
npm run preview
```

## Formulaire de contact

Le formulaire de contact utilise **Web3Forms**.
Il permet de recevoir les messages sans devoir mettre en place un backend spécifique.

Le formulaire collecte notamment :

* le nom ;
* l’adresse email ;
* le profil de la personne ;
* un message libre sur les attentes vis-à-vis de Radon Sense.
